# Petal-Drop
This game was created in 2023 for the final High School Computer Science project in 12th Grade.
It is a mimic of the popular game 'Hangman' with a little change.
With every wrong letter guess, petals drop one by one.
There are 5 chances of guesses, and 3 categories - Animals, Programming and Fruits; to choose from.
Score updates with each correct guess.
